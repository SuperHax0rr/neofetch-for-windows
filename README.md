## Features
- Displays Windows system information
- ASCII art output
- Custom color themes
- Terminal customization
- Launches instantly using a custom AutoHotkey keyboard shortcut

## Usage

Run the script directly:

```bash
python neofetch4windows.py
```

Or launch it instantly using the configured AutoHotkey shortcut.

## AutoHotkey Integration

An AutoHotkey script is included to launch NeoFetch for Windows with a global keyboard shortcut, allowing quick access similar to the original NeoFetch experience on Linux.
